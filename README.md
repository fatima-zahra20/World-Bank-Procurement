# 🌍 The Big Question: *Are loans actually worth it for developing countries?*

The contracts table alone already hints at a critical sub-question:

> If a developing country borrows $100M from the World Bank, but 80% of the contracts go to
> **foreign suppliers** — the money flows right back out of the country. The country is left
> with **the debt but not the economic activity.**

---

## How Each Table Contributes

| Table | What it adds to the question |
|---|---|
| **loans** | How much was borrowed, how much repaid, how much cancelled — *was the loan even used well?* |
| **contracts** | Who got the money — local or foreign suppliers? Which sectors? |
| **countries** | Income level, lending type — *are the poorest countries getting the worst deals?* |
| **education** | Did school enrollment actually improve after education loans? *Did the loan change anything on the ground?* |

---

## The Core Tension

By the end of this project we won't just have clean data — we'll have a **multi-angle verdict**
on whether these loans help or just create dependency and debt.
